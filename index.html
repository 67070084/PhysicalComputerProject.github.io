ระบบควบคุมมอเตอร์ผ่าน Wi-Fi ด้วย Arduino UNO R4

1. ความเป็นมาและจุดประสงค์

โครงงานนี้จัดทำขึ้นเพื่อสร้างอุปกรณ์ที่สามารถควบคุมมอเตอร์ผ่านเครือข่าย Wi-Fi ได้ โดยใช้บอร์ด Arduino UNO R4 WiFi เป็นตัวควบคุมหลัก และใช้หน้าเว็บในการสั่งงาน จุดประสงค์คือเพื่อเรียนรู้การทำงานของ Web Server บนไมโครคอนโทรลเลอร์ และเข้าใจการเชื่อมต่อระหว่างฮาร์ดแวร์กับอินเทอร์เฟซบนเว็บ

2. วิธีดำเนินงาน

  • ศึกษาโค้ดและตัวอย่างการเชื่อมต่อ Wi-Fi ด้วยบอร์ด Arduino
  • ออกแบบหน้าเว็บควบคุมมอเตอร์ (HTML, CSS, JavaScript)
  • เขียนโค้ด Arduino เพื่อรับคำสั่งจากหน้าเว็บ
  • เชื่อมต่อและทดสอบการควบคุมจริง
  • ปรับปรุงระบบแสดงสถานะและเพิ่มฟังก์ชัน Reset

3. สิ่งที่ได้ลองทำและผลการทดสอบ

- สามารถควบคุมการหมุนของมอเตอร์ผ่านหน้าเว็บได้จริง
- ปุ่ม Start, Stop และ Speed ทำงานได้ตามที่ออกแบบ
- หน้าเว็บแสดงสถานะการทำงานแบบเรียลไทม์
- มีการทดสอบทั้งบนคอมพิวเตอร์และมือถือ

4. อุปสรรคและการแก้ไข

  • หน้าเว็บไม่แสดงผล — ตรวจสอบโค้ด HTML และเพิ่ม server.begin()
  • ปุ่มไม่ตอบสนอง — ตรวจสอบ pinMode() และ debounce ของปุ่ม
  • เชื่อมต่อ Wi-Fi ไม่ได้ — ตรวจสอบ SSID/Password และระยะสัญญาณ

5. ผลลัพธ์และสิ่งที่ยังทำไม่ได้

ผลลัพธ์ที่ได้คือสามารถควบคุมมอเตอร์ผ่าน Wi-Fi ได้ตามต้องการ แต่ระบบยังไม่สามารถเชื่อมต่อจากอินเทอร์เน็ตภายนอกได้ (จำกัดในวง LAN) และยังไม่มีระบบล็อกอินหรือจำกัดสิทธิ์การใช้งาน

6. บทบาทของสมาชิกในโครงงาน

ชื่อสมาชิกหน้าที่ / สิ่งที่รับผิดชอบ
นายธีรัชเขียนโค้ด Arduino และตั้งค่า Wi-Fi Server
นางสาวศิรภัสสรออกแบบหน้าเว็บและตกแต่ง UI
นายอภิสิทธิ์ทดสอบระบบและแก้ไขปัญหาการเชื่อมต่อ
นางสาวปิยนุชจัดทำรายงานและสรุปผลโครงงาน

7. สรุปผลและแนวทางพัฒนา

โครงงานนี้สามารถทำงานได้ตรงตามเป้าหมาย คือควบคุมมอเตอร์ผ่าน Wi-Fi ได้จริง ในอนาคตสามารถพัฒนาต่อโดยเพิ่มระบบควบคุมผ่านอินเทอร์เน็ต (IoT), เพิ่มการแสดงผลสถานะด้วยกราฟ หรือสร้างแอปพลิเคชันสำหรับมือถือ
